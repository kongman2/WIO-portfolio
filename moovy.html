MOOVY

넷플릭스 트렌드를 기반으로 대화와 반응이 빠르게 축적되는 참여형 영화 리뷰 플랫폼

"기록 저장형 리뷰가 아닌, 트렌드 중심 커뮤니티 경험"

프로젝트 개요

넷플릭스 트렌드를 기반으로 한 참여형 영화 리뷰 플랫폼입니다. 사용자들이 영화에 대한 대화와 반응을 빠르게 공유하고, 트렌드 중심의 커뮤니티 경험을 제공합니다. 단순한 기록 저장이 아닌 실시간 소통과 트렌드 파악에 중점을 둡니다.

주요 기능

  • 넷플릭스 트렌드 기반 영화 추천
  • 실시간 리뷰 및 반응
  • 커뮤니티 토론 및 소통
  • 트렌드 분석 및 시각화

기술 스택

React · HTML · CSS · JavaScript
