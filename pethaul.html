배포 링크

무료 배포 환경 특성상 초기 로딩 지연 및
이미지 업로드 오류가 발생할 수 있습니다.

핵심 MVP 전략

"설문 → 진단 → 추천" 단일 흐름으로 개인화 경험 제공
MVP 포인트 의도
반려동물 설문 UX 상세 입력 없이 성향 기반 추천
유형별 상품 큐레이션 과잉 탐색 줄이고 즉시 선택
결과 카피 & 일러스트 이해도 및 공감 강화
진행률 & 단계 UX 이탈률 최소화
공통 컴포넌트 구축 반응형 & 유지보수 효율
반응형 레이아웃 (Bootstrap 활용) 디바이스별 동일 경험

→ 정보 입력 과정을 간략화하고, 즉시 추천과 구매 행동으로 연결하는 흐름을 만드는 것

메인 → 반려동물 설문조사 → 성향 진단 → 상품 추천 → 주문 및 교환/반품

브랜드 경험

"Old Game, New Experience"

게임화된 경험 + 신뢰 기반 추천 UX를 통해 브랜딩 감성을 UX/UI에 반영했습니다. 졸업작품으로 만든 실물 굿즈(WIO 브랜드)의 디자인을 웹 서비스 UI와 동일한 브랜드 톤으로 확장하여 오프라인 브랜드를 UX 언어로 전환한 경험입니다.

결과 & 배운 점

  • 설문부터 주문까지 사용자 흐름 일관성 확보
  • 브랜드 톤앤매너를 UI 요소로 확장해 감성 경험 유지

초기 컴포넌트 기준 정의 부재로 중복 구현이 발생했으나, 이후 프로젝트부터 플로우 · IA 사전 정의 프로세스를 도입했습니다. 또한 시각적 요소가 단순 장식이 아닌 사용자의 이해 · 신뢰 · 행동을 결정하는 UX 자원임을 깨달았습니다.
